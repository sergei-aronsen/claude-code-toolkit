### Task 3: Add validate-integrations-catalog.py to files.scripts[]

```bash
python3 -c "
import json
with open('manifest.json') as f:
    m = json.load(f)
scripts = m.setdefault('files', {}).setdefault('scripts', [])
target = 'scripts/validate-integrations-catalog.py'
exists = any((s.get('path') if isinstance(s, dict) else s) == target for s in scripts)
if not exists:
    scripts.append({'path': target, 'sha256': None})  # sha256 computed at install time
with open('manifest.json', 'w') as f:
    json.dump(m, f, indent=2)
    f.write('\n')
print('scripts[] OK')
"
```

Verify schema matches existing entries (some may use plain string paths; conform).
