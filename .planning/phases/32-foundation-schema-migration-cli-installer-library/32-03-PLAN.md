- NO `read -N` (Bash 4+; BACKCOMPAT-01 forbids).
- NO blanket `>/dev/null 2>&1` over assertions — preserve stderr for assert-fail diagnostics.
- NO dependency on `claude` CLI being on PATH — use mock binaries via `TK_MCP_CLAUDE_BIN` for S13/S14.
