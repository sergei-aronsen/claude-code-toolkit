<objective>
Add `--collect-all` flag to `scripts/validate-release.sh`. Semantics:
- Runs every one of the 13 cells regardless of individual failures (no fail-fast).
- After all cells complete, emits an aggregated ASCII table (Cell | Pass | Fail | Status)
  plus a summary line `Matrix: X/13 cells passed, Y assertions passed, Z assertions failed`.
- Exits 0 if every cell passed; exits 1 if any cell had at least one failure
  (standard 0/1, no graded exit codes per D-11).
- Mutually exclusive with `--all`; passing both together yields a mutex error with exit 2 (D-14).
- Default `--all` behavior stays exactly as today — fail-fast, preserved sandbox path,
  `exit 1` at first red cell (D-12).

Purpose: CI + release-checklist users need to see the full matrix failure picture
in one run without re-invoking the runner per failing cell. Also keeps the door
open for Phase 11's styled-diff pass (UX-01) to color the same data layer without
semantic changes.

Output: `scripts/validate-release.sh` modified — new `collect_cell()`,
`print_aggregate_table()`, and three accumulator arrays; `--collect-all` case arm;
mutex guard.

    ```bash
    # REL-03 — collect_cell: run cell body, accumulate per-cell counts, do NOT exit on failure
    # Sibling of run_cell() at line 259-276; omits the exit 1 so --collect-all can continue.
    collect_cell() {
        local cell_name="$1" body_fn="$2"
        local bp=$PASS bf=$FAIL
        echo ""
        echo "${CYAN}━━ Cell: ${cell_name} ━━${NC}"
        "$body_fn"
        local cp cf
        cp=$((PASS - bp))
        cf=$((FAIL - bf))
        if [ "$cf" -gt 0 ]; then
            echo "${RED}FAIL: ${cell_name}: ${cf} assertion(s) failed${NC}" >&2
        else
            echo "${GREEN}PASS: ${cell_name}${NC}"
        fi
        _COLL_NAMES+=("$cell_name")
        _COLL_PASS+=("$cp")
        _COLL_FAIL+=("$cf")
    }
    ```

    **Step 1c — Add `print_aggregate_table()` immediately after `collect_cell()`:**

    ```bash
    # REL-03 — print aggregated ASCII table after --collect-all completes
    # Uses printf width specifiers (BSD-portable — avoids GNU column --table-columns)
    print_aggregate_table() {
        local i=0 total=${#_COLL_NAMES[@]} cells_ok=0
        echo ""
        printf "%-32s %4s %4s %6s\n" "Cell" "Pass" "Fail" "Status"
        printf "%-32s %4s %4s %6s\n" "--------------------------------" "----" "----" "------"
        while [ "$i" -lt "$total" ]; do
            local n="${_COLL_NAMES[$i]}" cp="${_COLL_PASS[$i]}" cf="${_COLL_FAIL[$i]}" st="PASS"
            if [ "$cf" -gt 0 ]; then
                st="FAIL"
            else
                cells_ok=$((cells_ok + 1))
            fi
            printf "%-32s %4d %4d %6s\n" "$n" "$cp" "$cf" "$st"
            i=$((i + 1))
        done
        echo ""
        echo "Matrix: ${cells_ok}/${total} cells passed, ${PASS} assertions passed, ${FAIL} failed"
    }
    ```
