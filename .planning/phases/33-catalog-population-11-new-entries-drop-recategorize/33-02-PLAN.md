### Task 1: Verify MCP package names

For each entry, verify candidate package names exist via `npm view <pkg>` or `WebFetch https://www.npmjs.com/package/<pkg>`:

- **stripe**: `@stripe/mcp` (likely Stripe official agent-toolkit), `@stripe/agent-toolkit`, then community `mcp-stripe`.
- **youtrack**: `@jetbrains/mcp-server-youtrack`, `mcp-youtrack`, `youtrack-mcp`. JetBrains has been publishing MCPs via npm.
- **linear**: `@linear/mcp` (Linear-official, recent), `linear-mcp`, `mcp-linear`.
- **jira**: `@atlassian/mcp-server`, `atlassian-mcp`, `mcp-atlassian`. Pick Atlassian-official for Cloud variant.
- **figma**: `figma-developer-mcp` (Framelink, popular community), `@figma/mcp-server` (if exists).

Skip Figma's official "Dev Mode MCP" (desktop-only) — pick the headless npm package.

If a package can't be verified after 3 candidates, leave entry OUT and document.
