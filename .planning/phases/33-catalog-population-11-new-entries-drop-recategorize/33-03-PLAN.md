## Tasks

### Task 1: Verify packages with extra scrutiny for unofficial entries

For `unofficial: true` entries, apply stricter currency check:

- **notebooklm**: candidates `mcp-server-notebooklm`, `notebooklm-mcp`, or wrap of `nlm` CLI as Python MCP. Verify (a) package exists, (b) updated within last 12 months, (c) > 100 weekly downloads (community traction signal). If all candidates fail any of these, skip the entry — better none than broken.
- **slack**: `@modelcontextprotocol/server-slack` was deprecated by Anthropic. Find current community fork: `mcp-slack`, `slack-mcp`, `@slack/mcp-server` (if Slack publishes). If only deprecated package available, document and consider hint-only entry.
- **telegram**: candidates `telegram-mcp`, `mcp-telegram`, `mcp-server-telegram`. Apply same currency + downloads check. Pin to specific version in `install_args` if community quality varies (e.g., `npx -y telegram-mcp@1.2.3`).

If notebooklm or telegram cannot be verified, skip them. Slack should ideally not be skipped — find a working fork.
