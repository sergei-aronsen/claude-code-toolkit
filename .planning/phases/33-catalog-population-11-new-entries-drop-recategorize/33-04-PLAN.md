### Task 3: Add CLI blocks to firecrawl, playwright, sentry

For each entry, add a `components.cli` block:

**firecrawl:**

```jsonc
"cli": {
  "detect_cmd": "firecrawl",
  "install": {
    "darwin": "npm install -g @mendable/firecrawl-cli",
    "linux": "npm install -g @mendable/firecrawl-cli"
  },
  "post_install_hint": "firecrawl --version  # verify install; set FIRECRAWL_API_KEY env"
}
```

Verify package `@mendable/firecrawl-cli` exists. If not, try `firecrawl-cli` or `@firecrawl/cli`. If no CLI package exists, leave firecrawl MCP-only (no CLI block) and document in SUMMARY.

**playwright:**

```jsonc
"cli": {
  "detect_cmd": "playwright",
  "install": {
    "darwin": "npm install -g @playwright/test && npx playwright install",
    "linux": "npm install -g @playwright/test && npx playwright install"
  },
  "post_install_hint": "playwright --version  # browsers installed to ~/.cache/ms-playwright/"
}
```

Note: `playwright install` downloads browsers (~100MB+). Hint flags this.

**sentry:**

```jsonc
"cli": {
  "detect_cmd": "sentry-cli",
  "install": {
    "darwin": "brew install getsentry/tools/sentry-cli",
    "linux": "curl -sL https://sentry.io/get-cli/ | INSTALL_DIR=$HOME/.local/bin bash"
  },
  "post_install_hint": "sentry-cli login  # or set SENTRY_AUTH_TOKEN env"
}
```

Note: `detect_cmd` is `sentry-cli` (binary name), not `sentry`. The `INSTALL_DIR=$HOME/.local/bin` env var prefix on linux makes the install user-space (no sudo).
