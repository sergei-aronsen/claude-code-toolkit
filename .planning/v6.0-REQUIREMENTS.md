**Status:** Draft
**Created:** 2026-05-06

**Implementation strategy:** Mixed mode (gsd-fast / gsd-quick / gsd-plan-phase per PR risk)

---

## Vision

Toolkit v6.0 transitions from standalone-or-complement (v5.x) to **overlay-only**: explicit complement on top of GSD v1.40 + Superpowers v5.1.0 + external tools (Morph + better-model + claude-context). Zero duplication with base plugins.

Three-layer architecture:

```text
TOOLKIT v6.0 (overlay, gap-filler, MCP installer, bridge)
    ↓ assumes installed
GSD + Superpowers (Claude Code plugins)
    ↓ recommends (optional)
External tools (Morph, better-model, claude-context)
```

Result: 60% size reduction (651KB → ~259KB), 0% file duplication with base plugins.
