# v4.9 Integrations Catalog — Milestone Summary

## What shipped

A unified **Integrations Catalog** combining MCP servers + companion CLIs under a single `--integrations` TUI page, replacing the v4.6 standalone `--mcps` page. 20 MCP servers + 8 companion CLIs across 10 categories, with category-grouped rendering, per-row status detection, `unofficial: true` confirm gate, `--mcp-only` / `--cli-only` mutex flags, and a closing per-component summary table.

Status: **code-complete on `main`**. Awaiting manual `git tag v4.9.0` per CLAUDE.md never-push-main invariant.
