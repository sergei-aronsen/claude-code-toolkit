- **Finding:** `quality.yml` `test-init-script` job runs only inline Laravel/Next.js detection. Never invokes `make test`. Tests 21-27 (entire uninstall suite) ran only locally.
- **Root cause:** Phase 20 D-16 wrongly assumed `test-init-script` runs `make test`.
- **Fix:** Added new step `Tests 21-27 — uninstall + banner suite (UN-01..UN-08)` to `validate-templates` job, executing all 7 test scripts inline.
- **Evidence:** `.github/workflows/quality.yml` lines 109-117.

### Gap 2 (WARNING, UN-05/UN-06): Makefile test target gap

- **Finding:** `test-uninstall-idempotency.sh` and `test-uninstall-state-cleanup.sh` existed on disk and passed locally but were absent from the Makefile `test:` target.
- **Fix:** Appended Test 26 (idempotency) and Test 27 (state-cleanup) to Makefile after Test 25 banner gate.
- **Evidence:** `Makefile` lines 130-135.

Both fixes shipped in a single commit prior to this audit report's `passed` verdict.
