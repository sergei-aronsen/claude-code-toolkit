## Key Decisions

| Decision | Rationale |
|----------|-----------|
| Plain copy over symlink | Allows per-CLI customization; drift handled via SHA256 + `[y/N/d]` |
| AGENTS.md (not CODEX.md) for Codex | OpenAI standard; locked at top-level domain layer |
| `write_state` 10-arg extension over `bridges[]` patch in callers | Atomic, single source of truth, backward-compatible (default `'[]'`) |
| `_bridge_state_file` honors caller's STATE_FILE | Allows project-local installs to keep bridges[] scoped to project |
| `--bridges` and `--no-bridges` mutex | Avoid ambiguity; mirrors `--no-bootstrap`/`--bootstrap-only` |
| Default Y for install prompt, default N for drift prompt | Install is additive, drift is destructive |
| TUI rows hidden when CLI absent | No greyed-out clutter; matches v4.6 RTK pattern |
| Aggregator test over rewrite | 3 phase tests already cover 50 assertions; aggregator is zero-cost |
