## Key Decisions

- **zh register:** Formal written Chinese (书面语). Fullwidth CJK punctuation `，。：（）` in prose. ASCII in code blocks. Space between CJK and Latin (e.g., `MCP 服务器`).
- **ja register:** です/ます polite form. Katakana for common tech terms (`トークン`, `フレームワーク`, `スキル`, `フック`). Proper nouns stay in Latin (`Claude Code`, `RTK`, `MCP`). Space between kanji/katakana and Latin.
- **ko register:** -습니다 formal (합쇼체). ASCII punctuation throughout (standard in Korean dev docs). Space between Korean and Latin (`MCP 서버`, `Claude Code 툴킷`).
- **Install Modes structure:** All 3 languages correctly have `## {설치 모드|インストールモード|安装模式}` at H2 with 3 H3 sub-modes, matching README.md's `## Install Modes` structure added in v4.0.
- **MCP split:** All 3 languages correctly split MCP into `### Global` and `### Per-project` H3s with 4-line global block and dbhub per-project block — verbatim from README.md.
