### Sentinel scheme (D-09)

Four sentinels, each a single-line HTML comment:
```text
<!-- v42-splice: callout -->
<!-- v42-splice: fp-recheck-section -->
<!-- v42-splice: output-format-section -->
<!-- v42-splice: council-handoff -->
```

Detection (per-file, before splice):
```bash
total=$(grep -cF '<!-- v42-splice:' "$f" 2>/dev/null || true)
# 0 -> splice; 4 -> skip; 1-3 -> error (D-09)
```

### Council handoff slot string (D-08, byte-exact)

The footer paragraph must include the U+2014 em-dash placeholder verbatim:
```text
_pending — run /council audit-review_
```

This is the slot Phase 15 mutates. Em-dash is U+2014 (decimal 8212), NOT hyphen-minus (0x2D), NOT en-dash (U+2013).

Tempfile MUST be in same directory as target (`"${f}.XXXXXX"`) so `mv` is an atomic rename within the same filesystem.

Replace the stub `insert_blocks()` from Task 1 with a real implementation. The function takes one argument (path to the prompt file) and rewrites it via tempfile+mv with all 4 blocks inserted.

**Implementation strategy:** Use a single awk pass to do all 4 insertions in one go. Awk is portable, handles multi-line range deletion + insertion, and avoids the GNU-only sed flags that fail on macOS BSD.

**Step 1: Detect file shape (numbered vs unnumbered)**:

```bash
insert_blocks() {
    local f="$1"
    local tmp
    tmp=$(mktemp "${f}.XXXXXX")
    # shellcheck disable=SC2064
    trap "rm -f '$tmp'" INT TERM

    local has_numbered_sections=0
    local existing_selfcheck_line=0
    local existing_selfcheck_num=""
    local max_section_num=0
    local existing_reportfmt_line=0

    if grep -qE '^## [0-9]+\.' "$f"; then
        has_numbered_sections=1
        max_section_num=$(grep -oE '^## [0-9]+\.' "$f" | grep -oE '[0-9]+' | sort -n | tail -1)
    fi

    # Find existing SELF-CHECK heading line + number (if any)
    local sc_line
    sc_line=$(grep -nE '^## ([0-9]+\.\s*)?SELF-CHECK' "$f" | head -1 || true)
    if [ -n "$sc_line" ]; then
        existing_selfcheck_line=${sc_line%%:*}
        existing_selfcheck_num=$(echo "${sc_line#*:}" | grep -oE '^## [0-9]+' | grep -oE '[0-9]+' | head -1 || true)
    fi

    # Find existing REPORT FORMAT / OUTPUT FORMAT heading line (any numeric prefix)
    local rf_line
    rf_line=$(grep -nE '^## ([0-9]+\.\s*)?(REPORT FORMAT|OUTPUT FORMAT|ФОРМАТ ОТЧЁТА)' "$f" | head -1 || true)
    if [ -n "$rf_line" ]; then
        existing_reportfmt_line=${rf_line%%:*}
    fi
```

**Step 4: Insertion logic — single awk pass that emits the rewritten file to `$tmp`**:

The four insertions in one awk pass, ordered by file position (top to bottom):

1. Insert callout block immediately AFTER the H1 line (line 1) and any blockquote/blank line that follows.
2. If `existing_selfcheck_line > 0`: replace the SELF-CHECK section (from heading line through next `^## ` or `^---$` or EOF) with `fp_block`. Else: insert `fp_block` immediately BEFORE the first existing report-format heading (`existing_reportfmt_line`) — or append to EOF if no report-format heading exists.
3. Insert `of_block` immediately AFTER the existing report-format section (find its end: next `^## ` after `existing_reportfmt_line`, or EOF). If no report-format section exists, append at EOF after `fp_block`.
4. Append `ch_block` at EOF (always last H2).

Implementation: write a multi-line awk program. Because the four insertion points are at different file positions, the cleanest approach is to use awk variables for the 5 line-number anchors (`callout_after`, `selfcheck_start`, `selfcheck_end`, `reportfmt_end`, `eof`) computed in shell, then have awk emit the file with insertions.

```bash
    # Compute SELF-CHECK section end line (next ^## or ^---$ after the heading, else EOF)
    local selfcheck_end_line=0
    if [ "$existing_selfcheck_line" -gt 0 ]; then
        selfcheck_end_line=$(awk -v start="$existing_selfcheck_line" '
            NR > start && (/^## / || /^---$/) { print NR; exit }
            END { if (!found) print NR + 1 }
        ' "$f")
    fi

    # Compute REPORT FORMAT section end line (next ^## after the heading, else EOF)
    local reportfmt_end_line=0
    if [ "$existing_reportfmt_line" -gt 0 ]; then
        reportfmt_end_line=$(awk -v start="$existing_reportfmt_line" '
            NR > start && /^## / { print NR; exit }
            END { if (!found) print NR + 1 }
        ' "$f")
    fi

    # Single awk pass: emit rewritten file with 4 insertions
    awk -v callout="$callout_block" \
        -v fp_block="$fp_block" \
        -v of_block="$of_block" \
        -v ch_block="$ch_block" \
        -v sc_start="$existing_selfcheck_line" \
        -v sc_end="$selfcheck_end_line" \
        -v rf_start="$existing_reportfmt_line" \
        -v rf_end="$reportfmt_end_line" '
        BEGIN { callout_emitted = 0 }
        # Block 1: callout — after H1 line (NR == 1)
        NR == 1 {
            print
            print ""
            print callout
            callout_emitted = 1
            next
        }
        # Block 2: replace SELF-CHECK body (skip lines from sc_start to sc_end - 1)
        sc_start > 0 && NR == sc_start {
            print ""
            print fp_block
            print ""
            in_skip = 1
            next
        }
        sc_start > 0 && in_skip && NR < sc_end { next }
        sc_start > 0 && in_skip && NR == sc_end { in_skip = 0 }
        # Block 3: insert OUTPUT FORMAT block AFTER report-format section ends
        rf_start > 0 && NR == rf_end - 1 {
            print
            print ""
            print of_block
            print ""
            of_emitted = 1
            next
        }
        { print }
        END {
            # Block 2 fallback: no SELF-CHECK existed and rf_start > 0 — already covered if sc was inserted there
            # In practice, if sc_start == 0 (no existing SELF-CHECK), we insert fp_block before rf_start. Handle below.
            if (sc_start == 0 && rf_start == 0) {
                # Neither section exists — append fp_block + of_block at EOF
                print ""
                print fp_block
                print ""
                print of_block
            }
            # Block 3 fallback: rf_start == 0 (no report-format section) and of not yet emitted
            if (!of_emitted && rf_start == 0) {
                # of_block already emitted in the previous fallback if both missing; otherwise append now
                if (sc_start > 0) {
                    print ""
                    print of_block
                }
            }
            # Block 4: Council Handoff (always last)
            print ""
            print ch_block
        }
    ' "$f" > "$tmp"
```

NOTE: the awk above is the contract; the executor MAY refactor for readability so long as the 4 insertion points and ordering are preserved and all 4 sentinels appear in the output exactly once. The third insertion (OUTPUT FORMAT before EOF) needs a special case for files where `existing_selfcheck_line > 0` but `existing_reportfmt_line == 0` — in that case append `of_block` after the file's regular content, before the Council Handoff.
