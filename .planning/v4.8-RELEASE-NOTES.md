# v4.8.0 — Multi-CLI Bridge

Bridges `CLAUDE.md` to `GEMINI.md` (Gemini CLI) and `AGENTS.md` (OpenAI Codex CLI) so users running multiple agentic CLIs maintain a single source of truth.

## Highlights

- **New `scripts/lib/bridges.sh` (~700 lines)** — shape-A CLI detection (`is_gemini_installed`, `is_codex_installed`), banner injection, atomic state tracking, drift prompts. Bash 3.2 portable; no associative arrays.
- **`update-claude.sh` sync engine** — in-sync / orphaned / drift / skip decision tree, `[y/N/d]` prompt mirroring v4.3 UN-03 contract.
- **New flags:** `--no-bridges`, `--bridges <list>`, `--break-bridge <target>`, `--restore-bridge <target>`. `TK_NO_BRIDGES=1` env knob honored everywhere.
- **Install-time TUI rows** for detected CLIs in `install.sh`; conditional surface that hides cleanly when no CLI is present.
- **`docs/BRIDGES.md`** (168 lines, 9 sections) covering supported CLIs, plain-copy semantics, drift handling, symlink-vs-copy rationale, uninstall, future scope.

## Critical fixes during review

- **Phase 29 WR-01** — `is_protected_path` was silently keeping modified bridges. `BRIDGE_PATHS` bypass added to `prompt_modified_for_uninstall`.
- **Phase 29 WR-02** — bridges helpers ignored caller's `STATE_FILE` and always wrote to `$HOME/.claude/toolkit-install.json`, so sync silently no-op'd in real installs. Fixed `_bridge_state_file()` / `_bridge_lock_dir()` to honor `TK_BRIDGE_HOME` → `STATE_FILE` → default chain.
- **Phase 30 WR-01** — `--bridges` force path used `|| true` swallowing failures. Now captures rc and warns with rc-specific message.
- **Post-merge bash portability fix** — replaced invalid `${#BRIDGE_PATHS[@]:-0}` with `${BRIDGE_PATHS+set}` set-test plus separate length test (CI Linux bash 5.x rejected the original).

## Install / upgrade

```bash
# Fresh install — bridges auto-detected and offered
bash <(curl -sSL https://raw.githubusercontent.com/sergei-aronsen/claude-code-toolkit/main/scripts/init-claude.sh)

# Existing install — bridges sync on next update
bash <(curl -sSL https://raw.githubusercontent.com/sergei-aronsen/claude-code-toolkit/main/scripts/update-claude.sh)

# Opt out
bash <(curl -sSL ...) --no-bridges
TK_NO_BRIDGES=1 bash <(curl -sSL ...)

# Force-create when CLI not detected (rare)
bash <(curl -sSL ...) --bridges gemini,codex
```
