🔌 Stripe API key required

Payments + subscriptions — query, create test customers, debug charges. repo ↗

1. Get a restricted API key

  1. Open dashboard.stripe.com/apikeys/create.
  2. Pick Restricted key (NOT the secret key).
  3. Grant only the resources you want the AI to read/write — start with read-only on customers, subscriptions, charges.
  4. Use the test-mode key first; switch to live only after you trust the prompts.

2. Set the env var

Project-scope (default — lands in .env):

echo 'STRIPE_SECRET_KEY=rk_test_…' >> .env
echo '.env' >> .gitignore

3. Test

list the last 10 customers in test mode
Never use the unrestricted secret key (sk_live_…) with an AI agent. Restricted keys can be revoked granularly; secret keys can move money.
