🔌 PostHog API key required

Product analytics + error tracking + LLM cost tracking + feature flags. docs ↗

1. Get a personal API key

  1. Open app.posthog.com/settings/user-api-keys.
  2. Click Create personal API key.
  3. Scope: pick the projects + capabilities you want the AI to read (start with read-only).
  4. Copy the phx_… token.

2. Set the env var

Format MUST be Bearer <token> (the MCP sends it verbatim as the Authorization header):

export POSTHOG_AUTH_HEADER="Bearer phx_…"

Persist:

echo 'export POSTHOG_AUTH_HEADER="Bearer phx_…"' >> ~/.zshrc
source ~/.zshrc
