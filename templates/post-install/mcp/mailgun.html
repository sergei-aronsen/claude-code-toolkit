🔌 Mailgun API key required

Transactional email — send, validate, query stats, manage suppressions. repo ↗

1. Get the API key

  1. Open app.mailgun.com/app/account/security/api_keys.
  2. Click Add new key → name it claude-mcp.
  3. Copy the value — Mailgun shows it once.

2. Set env vars (project-scope by default)

echo 'MAILGUN_API_KEY=your-key-here' >> .env
echo 'MAILGUN_API_REGION=us' >> .env  # or `eu` if you're on the EU plan
echo '.env' >> .gitignore

3. Test

send a test email to me@example.com from the verified domain
