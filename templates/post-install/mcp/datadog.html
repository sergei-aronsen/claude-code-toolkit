1. EU users — edit the URL

Default URL is the US region. If your Datadog account lives on datadoghq.eu, edit ~/.claude.json and change the URL:

jq '.mcpServers.datadog.url = "https://app.datadoghq.eu/api/v2/mcp/sse"' ~/.claude.json > ~/.claude.json.tmp \
  && mv ~/.claude.json.tmp ~/.claude.json

2. Authenticate

Restart Claude Code in any project. The first MCP call opens a browser tab — log into Datadog and approve the OAuth scope. The token is stored automatically.

3. Test

show me the p99 latency of the api-gateway service in the last hour
Cost discipline. Datadog billing is per-metric + per-host. Heavy use of the MCP from an AI agent can blow your monthly budget — set custom-metric quotas in the Datadog UI before letting Claude loose.
