📊 Rate-limit Statusline no setup needed

macOS-only. Reads the Claude Code OAuth token from Keychain and shows session + weekly usage in your status bar.

Verify

Open Claude Code in any terminal — the statusline shows session usage in real time. If it stays blank, force a refresh:

bash ~/.claude/statusline-refresh.sh
macOS only. The statusline uses the macOS Keychain to read the OAuth token. Linux / Windows are not supported by this component.
