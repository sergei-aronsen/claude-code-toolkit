🌉 Multi-CLI Bridges no setup needed

Auto-sync project CLAUDE.md → GEMINI.md (Gemini CLI) and AGENTS.md (OpenAI Codex CLI). Drift-detected on every install.

Verify

The bridges write GEMINI.md / AGENTS.md next to your CLAUDE.md. Open them in your project root and check the header line:

head -3 GEMINI.md AGENTS.md 2>/dev/null

Re-sync after a CLAUDE.md edit

The next install / /update-toolkit run picks up changes automatically. To force-refresh now:

bash <(curl -sSL https://raw.githubusercontent.com/sergei-aronsen/claude-code-toolkit/main/scripts/install.sh) --no-banner --bridges-only
Symlink users: if you symlinked GEMINI.md / AGENTS.md to CLAUDE.md, the bridges defensively skip — they will not overwrite your symlink. Remove the symlink to enable bridge-managed copies.
