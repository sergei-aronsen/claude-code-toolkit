⚖️ Supreme Council component

Multi-AI plan review (/council). Sends a plan to Gemini and ChatGPT in parallel for independent review.

Pick one route

Council needs either the official CLI binaries or direct API keys. CLI route is cheaper (uses your existing CLI subscription); API route is faster to set up.

Route A — via CLI binaries (recommended)

  1. Install Gemini CLI: npm install -g @google/gemini-cli
  2. Install OpenAI Codex CLI: npm install -g @openai/codex-cli (or use any equivalent CLI you have).
  3. Run each once interactively to authenticate (gemini opens an OAuth flow; codex needs codex login).
  4. Council auto-detects both and uses them. No API keys needed.

Route B — direct API keys

Get the keys:

  • Gemini — aistudio.google.com/apikey
  • OpenAI — platform.openai.com/api-keys

Set them in ~/.claude/council/config.json (auto-created on first run):

{
  "gemini_api_key": "AIza…",
  "openai_api_key": "sk-…",
  "models": {
    "gemini": "gemini-3-pro-preview",
    "openai": "gpt-5.2"
  },
  "cost_gate_usd": 0.50
}

Test

brain "ship a feature flag for new pricing tiers"

Or from inside Claude Code:

/council ship a feature flag for new pricing tiers
Cost gate. Council estimates the round-trip cost before each call and refuses if it exceeds cost_gate_usd (default $0.50). Bump it for larger plans.
