🧠 claude-memo component

Persistent engineering memory — vault + 4 hooks. repo ↗

1. Reload your shell

The installer added MEMO_VAULT_PATH to your shell rc. Pick it up in the current terminal:

source ~/.zshrc   # or open a new terminal

2. Optional: enable auto-classification (~$0.002 per session)

Without this, manual /memo still works for free. With it, every session is auto-summarised by Claude Haiku and saved as a structured note.

Get an API key at console.anthropic.com/settings/keys, then export:

export ANTHROPIC_API_KEY=sk-ant-…

Persist it in your shell rc:

echo 'export ANTHROPIC_API_KEY=sk-ant-…' >> ~/.zshrc
source ~/.zshrc

3. Open the vault in Obsidian (optional, recommended)

  1. Install Obsidian (free).
  2. Launch Obsidian → Open folder as vault.
  3. Pick ~/memo-vault and click Open.
  4. Enable the Graph view for [[wikilink]] navigation.

4. Test it

Restart Claude Code in any project directory. The SessionStart hook injects relevant vault notes. Save a new note from inside Claude Code:

/memo
Disable auto-classification later? Just unset ANTHROPIC_API_KEY (and remove from rc). All hooks stay; only the Haiku call is gated by the env var.
